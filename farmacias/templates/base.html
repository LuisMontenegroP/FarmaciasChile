FarmaciasChile

Proyecto Django Taller Integrado

FarmaciasChile es una Aplicación de farmacias que mostrará al usuario la ubicación del establecimiento farmaceutico más cercano, además este podrá realizar un filtro de éste de su preferencia, para tener una vista personalizada.
